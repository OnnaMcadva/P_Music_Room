### Стек проекта:
*   **Бэкенд:** Python (FastAPI) + PostgreSQL + Redis.
*   **Клиент (Android, iOS, Web):** Kotlin Multiplatform (KMP) + Compose Multiplatform.

---

### РАСПРЕДЕЛЕНИЕ РОЛЕЙ НА 4 ЧЕЛОВЕК

#### **Участник 1: Бэкенд на Python (Главный по пайтону ^-^)**
*Обязанности: База данных, REST API, авторизация и безопасность.*
*   **Что делает:**
    1.  Проектирует БД (PostgreSQL) для пользователей, плейлистов и треков.
    2.  Пишет авторизацию: Email/пароль (с подтверждением почты) + OAuth (Google/Facebook).
    3.  Реализует REST API на **FastAPI** с автоматической документацией **Swagger** (требование ТЗ).
    4.  Обеспечивает безопасность: JWT-токены, защита от брутфорса, разграничение прав (юзер видит только свои данные).
    5.  *Бонус:* Логика платной/бесплатной подписки (ограничение функционала на уровне API).

#### **Участник 2: Реал-тайм Бэкенд & DevOps (Помощник бэкендера, тоже на Python)**
*Обязанности: WebSockets, конкуренция, логирование и нагрузочное тестирование.*
*   **Что делает:**
    1.  Реализует **WebSockets** на FastAPI для мгновенного обновления очереди треков и голосования.
    2.  Решает проблему конкуренции (race conditions) через **Redis** (чтобы сервер не падал, когда 100 человек одновременно голосуют за один трек).
    3.  Настраивает логирование всех действий с клиента (Платформа, Устройство, Версия приложения) в БД/файлы.
    4.  Проводит нагрузочное тестирование (раздел Ramp-up) с помощью **Locust** (инструмент на Python).
    5.  Упаковывает бэкенд в Docker и настраивает CI/CD.

#### **Участник 3: Kotlin-разработчик №1 (Интерфейс, Навигация и Авторизация)**
*Обязанности: Архитектура KMP, UI/UX на Compose Multiplatform, профили.*
*   **Что делает:**
    1.  Настраивает проект **Kotlin Multiplatform (KMP)** под Android, iOS и Web.
    2.  Верстает общий интерфейс на **Compose Multiplatform** (экраны входа, регистрации, профиля пользователя, настроек приватности).
    3.  Интегрирует библиотеки для входа через Google и Facebook на клиенте.
    4.  Реализует навигацию между экранами внутри приложения.
    5.  *Бонус (Мультиплатформенность):* Собирает и тестирует проект под Web (в KMP это компилируется в JS/Wasm).

#### **Участник 4: Kotlin-разработчик №2 (Интеграция API, Офлайн-режим и IoT)**
*Обязанности: Сетевой клиент (Ktor), WebSockets на клиенте, локальная БД, датчики.*
*   **Что делает:**
    1.  Настраивает сетевой клиент **Ktor** (на Kotlin) для запросов к Python-бэкенду.
    2.  Подключает WebSockets на клиенте для реал-тайм фич (голосование за треки, совместный плейлист).
    3.  Интегрирует аудио-плеер для проигрывания музыки.
    4.  *Бонус (Офлайн-режим):* Настраивает локальную БД **SQLDelight** (на Kotlin) для сохранения данных без интернета и пишет логику синхронизации при подключении к сети.
    5.  *Бонус (IoT):* Пишет интеграцию с Bluetooth/iBeacon на Kotlin для отправки пушей при приближении к гео-событию.
